🔄 محول JSON ↔ YAML ↔ TOML

محول JSON و YAML و TOML المجاني: تحويل بين صيغ البيانات الشائعة أونلاين. يدعم التحويل في أي اتجاه مع معالجة الأخطاء ونسخ النتيجة بضغطة زر.

مشاركة: فيسبوك تويتر واتساب
⟷

ما هو محول JSON ↔ YAML ↔ TOML؟

محول JSON ↔ YAML ↔ TOML هو أداة مجانية للتحويل بين صيغ البيانات الشائعة المستخدمة في البرمجة وتطوير الويب. يدعم التحويل في أي اتجاه بين الصيغ الثلاث: JSON (JavaScript Object Notation)، YAML (YAML Ain't Markup Language)، وTOML (Tom's Obvious Minimal Language). كل صيغة لها مزاياها واستخداماتها، وهذه الأداة تساعدك على التحويل بينها بسهولة.

كيف تستخدم المحول؟

استخدام الأداة بسيط: اختر صيغة الإدخال (JSON، YAML، أو TOML) من القائمة المنسدلة الأولى، اختر صيغة الإخراج من القائمة الثانية، الصق البيانات في مربع الإدخال، ثم اضغط "تحويل". يمكنك استخدام زر "تبديل" لعكس اتجاه التحويل بسرعة. النتيجة تظهر فوراً مع إمكانية نسخها بنقرة واحدة.

أنواع البيانات المدعومة

  • سلاسل نصية (Strings): نصوص عادية ومقتبسة
  • أرقام (Numbers): أعداد صحيحة وعشرية
  • قيم منطقية (Booleans): true/false
  • مصفوفات (Arrays): قوائم مرتبة من القيم
  • كائنات (Objects): مجموعات من أزواج المفتاح-القيمة
  • قيم فارغة (Null): null/None

الأسئلة الشائعة

ما هو محول JSON ↔ YAML ↔ TOML؟

محول JSON ↔ YAML ↔ TOML هو أداة مجانية للتحويل بين صيغ البيانات الشائعة. يدعم التحويل في أي اتجاه: JSON إلى YAML، YAML إلى JSON، JSON إلى TOML، TOML إلى JSON، YAML إلى TOML، TOML إلى YAML.

كيف أحول JSON إلى YAML؟

اختر صيغة الإدخال JSON وصيغة الإخراج YAML، الصق كود JSON في مربع الإدخال، ثم اضغط تحويل. ستحصل على كود YAML المقابل.

ما الفرق بين JSON و YAML و TOML؟

JSON صيغة خفيفة لتبادل البيانات تستخدم في APIs. YAML صيغة أكثر قابلية للقراءة البشرية تستخدم في ملفات الإعدادات. TOML صيغة بسيطة تركز على سهولة القراءة وتستخدم في إعدادات المشاريع مثل Rust (Cargo.toml) وPython (pyproject.toml).

هل يدعم المحول التعليقات؟

نعم، عند التحويل من YAML أو TOML إلى JSON، يتم تجاهل التعليقات لأن JSON لا يدعمها. عند التحويل من JSON إلى YAML أو TOML، لا تتم إضافة تعليقات تلقائياً. يمكنك إضافة التعليقات يدوياً بعد التحويل.
